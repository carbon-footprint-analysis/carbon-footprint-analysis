# \# Carbon Footprint Analysis

# 

# \## 📊 Overview

# This project aims to estimate and analyze CO₂ emissions from corporate fleets, comparing combustion and electric vehicles.

# 

# \## 🎯 Objective

# Provide insights to support sustainable decision-making using data analysis and machine learning.

# 

# \## 🏗️ Project Structure

# \- `docs/` → project documentation (CRISP-DM, architecture, decisions)

# \- `data/` → raw and processed datasets

# \- `notebooks/` → exploratory analysis

# \- `src/` → reusable code (data, features, models, pipeline)

# \- `wrapper/` → API layer

# \- `frontend/` → interface
